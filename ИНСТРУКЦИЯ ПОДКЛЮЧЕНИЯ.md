# Supabase MCP — подключение к Claude Desktop

Токен не нужен. Аутентификация через браузер при первом запуске.

---

## Установка

**1. Запустить скрипт** из терминала в папке проекта:
```bash
bash setup-supabase-mcp.sh
```

**2. Выбрать `1. Установить`**

**3. Получить MCP URL проекта:**
- Открыть [supabase.com](https://supabase.com) → выбрать проект
- Нажать **Connect** (кнопка в шапке)
- Вкладка **App Frameworks → MCP**
- Скопировать URL вида:
  ```
  https://mcp.supabase.com/mcp?project_ref=XXXXX
  ```
  *(или всю команду `claude mcp add ...` — тоже подойдёт)*

**4. Вставить URL** в терминал → ввести имя сервера (или нажать Enter — подставится автоматически)

**5. После перезапуска Claude Desktop** откроется браузер → подтвердить авторизацию Supabase

✅ Готово. Сервер виден в Claude: молоток → список подключений.

---

## Удаление

```bash
bash setup-supabase-mcp.sh
```

Выбрать `2. Удалить` → выбрать номер сервера из списка.

Удаляются только серверы, установленные этим скриптом. Остальные не затрагиваются.

---

## Примечания

- Конфиг Claude Desktop: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Перед каждым изменением создаётся бэкап с временной меткой
- Один скрипт — один проект Supabase (разные проекты = разные имена серверов)
